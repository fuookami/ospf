# Quickstart

## Installation

### Kotlin / Maven

**Version Requirements:**

* JDK: 17+ or 8+
* Maven: 3+

ospf-kotlin has been published to the Maven Central Repository. If using Maven, simply add the corresponding dependencies in the pom.xml file:

```xml
<dependency>
    <groupId>io.github.fuookami.ospf.kotlin</groupId>
    <artifactId>ospf-kotlin-starter</artifactId>
    <version>1.0.71</version>
</dependency>

<dependency>
    <groupId>io.github.fuookami.ospf.kotlin</groupId>
    <artifactId>ospf-kotlin-starter-jdk8</artifactId>
    <version>1.0.71</version>
</dependency>
```
